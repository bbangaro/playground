## Description

프론트 개발만 주구장창 하다가 감 잡으려고 공부하는 [Nest](https://github.com/nestjs/nest) framework TypeScript

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
